# MUNICIMAP

**3D Interactive Map with Wayfinding System for Olongapo City Hall**

## Overview

MUNICIMAP is a 3D interactive mapping system designed to help visitors easily navigate inside Olongapo City Hall. It provides a visual and user-friendly wayfinding experience, reducing confusion and improving accessibility for both first-time and returning visitors.

## Features

* 3D interactive map visualization
* Wayfinding system for navigating offices and locations
* Real-time user interaction within the map
* Built using modern web technologies

## Setup Guide

### Prerequisites

Install Node.js(https://nodejs.org/en/download)

Node.js is required to manage dependencies and run the development server.

### Installation

Open a the folder in VSCODE and then open terminal and install the required dependencies:

#### Install Three.js

```
npm install --save three
```

#### Install Vite (Build Tool)

```
npm install --save-dev vite
```

### Run the Project

Start the development server by running:

```
npx vite
```

If everything is set up correctly, a local development URL will appear in your terminal, such as:

```
http://localhost:5173
```

Open that URL in your browser to view the application.

## Tech Stack
